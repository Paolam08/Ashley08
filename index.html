Fundamentos de CSS

1. Color de texto

El color del texto se puede cambiar con la propiedad color.

Este texto es de color rojo.

2. Color de fondo

Se cambia usando background-color.

Este div tiene fondo amarillo.

3. Tamaño de fuente

Se usa la propiedad font-size.

Este texto es más grande.

4. Bordes

CSS permite agregar bordes con border.

Este div tiene un borde verde punteado.

5. Márgenes

Los márgenes externos se ajustan con margin.

Este div tiene un margen amplio.

6. Relleno (Padding)

El espacio interno se ajusta con padding.

Este div tiene relleno interno.

7. Fuente

La apariencia de la letra se controla con font-family.

Este texto usa una fuente de máquina de escribir.
